# Plantilla API + Base de Datos Relacional

Este proyecto es una plantilla para conectar Node.js (Express) con MySQL.

## Contenido
- app.js → Servidor Express y rutas API
- db.js → Conexión a MySQL
- bd_relacional.sql → Script SQL para crear base de datos
- guion.txt → Guía para sustentar el proyecto

## Requisitos
- Node.js instalado
- MySQL instalado

## Instalación
1. Importa el archivo `bd_relacional.sql` en MySQL.
2. Instala dependencias:
   npm install express mysql2
3. Ejecuta el servidor:
   node app.js
